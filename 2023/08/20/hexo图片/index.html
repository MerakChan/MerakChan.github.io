hexo如何上传图片

发表于 2023-08-20 更新于 2025-03-04 分类于 hexo使用

1. 本地静态图片上传

方法1：全局资源文件夹

在hexo文件夹下的source目录下，新建一个文件夹叫images(名字随意)，将要插入的图片放在该文件夹中。
然后使用md文档插入图片的格式使用相对路径进行插入即可（/image/xxx.png “name”）。

2. 使用图床

如果不想将图片保存在本地，可以使用图床（如 GitHub 图床、SM.MS、Imgur）来管理图片。

使用 GitHub 作为图床仓库 并使用 PicGo 上传图片

  • 创建github仓库，勾选public

  • 创建token（填写用途，并勾选repo），复制备份

  • 下载PicGo，图床设置->github->添加或者创建新的

  • 在PicGo上传区选择对应的仓库，上传图片生成链接，再用md语法添加图片即可。
    注意：要保证github上的仓库分支和picgo填写的分支名称一致

总结

  • 本地方式（source/images/）适用于简单博客，不需要额外插件。
  • 图床（GitHub、SM.MS、Imgur）适用于减少博客体积，提升访问速度。
  • CDN 适用于高级用户，有自己的服务器或云存储。
